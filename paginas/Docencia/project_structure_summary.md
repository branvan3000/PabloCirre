# Resumen de Estructura de Proyectos y Hitos Profesionales

Este documento resume la estructura de directorios y los proyectos clave identificados en el ecosistema digital de Pablo Cirre.

## 1. Big Data & OpenData (`/paginas/OpenData`)

El núcleo tecnológico más extenso, centrado en la agregación y estructuración de datos empresariales B2B.

* **Centraldecomunicacion.es**:
  * **Foco**: Sector Turístico en España.
  * **Volumen**: 1.4 Millones de empresas.
  * **Tecnología**: Arquitectura de datos masiva, limpieza y enriquecimiento.
* **CompaniesData.cloud**:
  * **Foco**: Datos empresariales a nivel europeo/global (27 países).
  * **Modelo**: API REST para consultas en tiempo real y descargas masivas.
* **UKBusinessDatabases**:
  * **Foco**: Mercado de Reino Unido (840K+ empresas).
  * **Valor**: 100% de registros con email verificado.
